## CATEGORY 1: JOB POSTINGS

### Prompt 1: Job Description Builder
```
Create a complete job description for [job title] at [company type].

Include:
- Job title
- Summary (3-4 sentences)
- Key responsibilities (6-10)
- Required qualifications
- Preferred qualifications
- Education requirements
- Experience level
- Salary range (optional)
- Benefits highlights
- Location (remote/hybrid/onsite)
- How to apply

Company culture: [describe]
```

### Prompt 2: Job Ad Generator
```
Create a compelling job ad for [position].

Platform: [LinkedIn/Indeed/Monster]

Include:
- Attention-grabbing headline
- Opening hook
- Role highlights (3-5 bullet points)
- Must-have requirements
- What makes this opportunity great
- Call to action with application link

Target candidate: [describe ideal candidate]
```

### Prompt 3: Skills Assessment Creator
```
Create a skills assessment for [job title].

Include:
- Technical skills to test (list 10-15)
- Practical exercises
- Case studies/scenarios
- Scoring criteria
- Time allocations
- How to evaluate results

Role level: [entry/mid/senior]
Industry: [describe]
```

### Prompt 4: Compensation Benchmark Request
```
Create a compensation research request for [job title].

Include:
- Role overview
- Key responsibilities
- Required experience
- Skills needed
- Location/remote status
- Market range needed
- Company size/type
- Equity/bonus considerations

Gather data from: [sources]
```

### Prompt 5: Internal Job Posting
```
Create an internal job posting for [position] for employees.

Include:
- Position overview
- Why to apply (benefits/growth)
- Requirements
- Application process
- Timeline
- Who to contact with questions

Level: [same level/lateral/promotion]
```
