## 💡 Sample Prompts

### Job Description Builder
```
Create a complete job description for a [job title] position at a 
[company type/size]. 

Include:
1. Job summary (3-4 sentences)
2. Key responsibilities (6-8 bullet points)
3. Required qualifications
4. Preferred qualifications
5. Salary range placeholder
6. Benefits highlight
7. Company culture description
8. Application instructions
```

### Interview Questions Generator
```
Generate 15 interview questions for hiring a [job title] that:
- Assess both technical skills and cultural fit
- Include 5 situational/behavioral questions
- Have 5 questions testing problem-solving
- Include 5 questions about [specific skill/quality]
- Are not cliché or overused
```
