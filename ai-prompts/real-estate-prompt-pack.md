## 💼 Sample Prompts

### Property Listing Generator
```
Write a compelling property listing description for a [bedroom/bathroom] 
[home type] located in [neighborhood/city]. 

Features include: [list key features]
Target buyers: [describe ideal buyer]

Make it:
- 150-200 words
- Use sensory language (sights, sounds, smells)
- Highlight unique selling points
- Include neighborhood amenities
- End with a call to action
```

### Lead Follow-up Script
```
Create a 5-touch lead follow-up sequence for real estate leads 
who have shown interest in [buying/selling/renting] in [target area].

Include:
- Initial outreach (within 24 hours)
- Follow-up #2 (3 days later)
- Follow-up #3 (1 week later)
- Follow-up #4 (2 weeks later)
- Final "I'm here when ready" message

Make each message:
- Under 100 words
- Personal and conversational
- Include value in each touch
- Clear call to action
```
